# Assets

## avatar.jpg

Сюда положи свою фотографию **с именем `avatar.jpg`** (та самая с акулами).

- Оптимальный размер: 1200×750 (соотношение 16:10)
- Формат: JPG, до ~500 КБ для быстрой загрузки
- Если файла нет — дашборд покажет fallback-эмоджи (🦈) на градиентном фоне

После того как положишь файл и сделаешь `git add assets/avatar.jpg && git commit && git push`, Vercel автоматически задеплоит, и фото появится в Mini App.
